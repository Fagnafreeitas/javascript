Tipos de saída em javascript

Vamos aprender as diferenças formas de exibir informações com javascrript

  1. 1. Exibir através de um elemento HTML usando getElementById

    Utilizamos getElementByIdpara acessar o elemento com um ID especifico.

    Em seguida, usamos textContent ou innerHTML para mostrar o conteúdo

  2. 2. Exibir com document.write()

    Este método escreve diretamente no corpo da página. Não é recomendado para uso moderno, mais aindda funciona.

  3. 3. Exibir com alert()

    Este comando exibe uma janela de alerta para o usuário.

  4. 4. Exibir com console.log()

    Este método exibe mensagens no console do navegador(f12 -> abaConsole?)
