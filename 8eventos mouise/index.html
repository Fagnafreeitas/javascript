Eventos Mouse

Nesta aula vamos entender o que é um evento mouse.

Eventos são ações disparadas pela interação dos úsuarios na página

E a aplicação desses eventos tornam as páginas interativas e dinâmicas .

  • onclick-> Disparando quando recebe um clique
  • ondblclick-> Disparando quando recebe um clique duplo
  • onmouseover-> Disparando quando quando o mouse esta sobre o elemento.
  • onmouseout-> Disparando quando o mouse sai do elemento.
  • onmousedown-> Quando o botão do mouse é pressionado.
  • onmouseup-> Quando o botão do mouse é solto
  • onfocus-> Quando o elemento recebe o foco
  • onblur-> Quando o elemento perde o foco
  • onchange-> Quando o valor de um campo é alterado
  • onkeypress->Quando a tecla é pressionada e solta.
  • onkeydonw-> Quando a tecla é pressionada e
  • onkeyup-> Quando uma tecla é solta
  • onreset->Quando um formulário é resetado.
