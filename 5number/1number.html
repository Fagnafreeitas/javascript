Tipos primitivos

    Tipo String O tipo string do javaScript é usado para representar dados textuais. Cada elemento na string ocupa uma posição na string. O primeiro elemento está no inicio o, próximo no indice o, oproximo no indice 1 e assim por diante. o comprimento de um string é o numero de elementos nela.

    ex; Azul, Carlos, idade.

  • Number

    • O tipo de Number é capaz de armazenar números de ponto flutuante positivos e negativos.

    Ex: 7, 15, -8, 1.5, 0,2, 7.9

  • Boolean

    • Boolean representa uma entidade lógica e pode ser dois valores: true e falso.

    ex: true false

  • Null

    • o tipo tem exatamente um valor : null valor nulo ou vazio.
